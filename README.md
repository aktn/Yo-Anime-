## About the Project

User can search for trailers and details of Anime through Kitsu API.
The project is created using React, TypeScript, Hooks, SASS & Styled components.

It can be checkout at [link](https://laughing-rosalind-009f10.netlify.com/)

In the project directory, you can run:

#### `npm install`

#### `npm start`

#### `npm test`

#### `npm run build`
